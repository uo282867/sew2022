Introducción a Ember.js

Ember.js es un framework de codigo abierto para el desarrollo web usando JavaScript que emplea un diseño componente-servidor.

Ember.js se ha empleado en aplicaciones bastante famosas como: Apple Music, LinkedIn, Twitch...

Enlace a la página oficial de Ember: emberjs.com

Bases iniciales de Ember.js

Segun los desarrolladores, fue creado entorno a las siguientes ideas:

Un desarrollo ambicioso

Ember se propone proporcionar una solución integral al problema de la aplicación del lado del cliente.

Más productivo desde el primer momento

Ember esta formado por un conjunto de herramientas que trabajan juntas para facilitar el desarrollo web.

Estabilidad sin estancamiento

Esta es la idea de que la compatibilidad con versiones anteriores es importante y se puede mantener mientras se sigue innovando y evolucionando el framework.

Previsión de futuros estándares web

Ember adoptó los estándares en torno a JavaScript y la web, incluidas las promesas, los componentes web y la sintaxis ES6.
